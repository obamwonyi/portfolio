Simple Register and Login with PHP

This project is a simple Register and Login sytem , its purpose is to solidify my knowledge on templating (which I built with PHP and did not use any dependency) and routing as well as autoloading with PHP with out any dependency, the only requirment for this project was psr-4 autoloading which was added with composer.

I used MariaDB for my database but I know it will also work with MYSQL as the syntax are very close.
Error message for empty values and invalid input was done with PHP, but for production I do use both.

The project is not hosted and can be found in my github repo.
